0. Requeriminetos

PMS está probado en Debian Squezee y Wheezy y Ubuntu Lucid. Necesita una tarjeta gráfica OpenGL y que los drivers para Xorg con aceleración por hardware estén correctamente instalados. Ejecutando "glxgears" en una terminal debería de abrir una ventana con tres engranajes girando. Si no es así, revisa la documentación de tu distribución.

1. Instala pd-extended y sus dependencias:
    
For pd-extended, you can use the apt repository. You can either add this line to the bottom of /etc/apt/sources.list Or you can add it in the GUI administration program Synaptic (in the Settings -> Repositories menu, then the Third-Party Software tab). Choose the line for your version of Debian or Ubuntu from the options below:

 # Debian/squeeze
 deb http://apt.puredata.info/releases squeeze main

 # Ubuntu/lucid 10.04 LTS
 deb http://apt.puredata.info/releases lucid main

2. Instala libartnet:

  	cd externals/libartnet-1.1.0
        ./configure --prefix=/usr
  	make
	sudo make install

3. Arrancando main.pd con pure data extended ya debería de estar funcionando. Dentro del programa puedes configurar más cosas como las direcciones DMX, subnet y universos artnet, las dimensiones de la pantalla y los textos para mostrar. Ver manual.txt para más detalles.
