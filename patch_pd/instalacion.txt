PMS sólo está probado en Debian Squezee, aunque debería funcionar en todos los Linux y con pocas modificaciones se podría hacer funcionar en MacOs X. Se dan instrucciones sólo para Debian.

1. Instala pd-extended y sus dependencias:
    
	apt-get install pd-extended

2. Descarga libartnet de la web del autor.

	wget http://linux-lighting.googlecode.com/files/libartnet-1.1.0.tar.gz

3. Descomprime el paquete en un directorio.

	tar -zxf artnet-1.1.0.tar.gz
	
4. Entra en el directorio.
  
  	cd artnet-0.x.y.tar.gz
	
5. Corre el script de configuración.
	
        ./configure
	
6. Construye los ficheros fuentes.
  
  	make
 
7. Conviértete en root
  	
	su
	(enter root password)

8. Instala las bibliotecas

  	make install

8.5 Instala Gem para control de textos, problema en las fuentes con paquete debian:

Añade las fuentes a /etx/apt/sources.list deb http://www.debian-multimedia.org squeeze main non-free
	apt-get install debian-multimedia-keyring
	apt-get update
	apt-get install
        apt-get build-dep puredata gem pdp
	apt-get install ttf-dejavu libfftw3-dev libspeex-dev libvorbis-dev libflac-dev libsndfile1-dev libsamplerate0-dev libgavl-dev libmad0-dev dssi-dev liblo-dev flite1-dev libtk-img tcllib ftgl-dev
	wget http://gem.iem.at/releases/0.92.3/gem-0.92-3.tar.gz
	Descomprime el tar y compila con:
		./configure
		make
	
	(Alternativamente a la compilación hemos compilado y subido a la web el Gem con soporte de textos - Puedes ahorrarte estos pasos si te lo descargas) 
	
	Haz una copia de seguridad o renombra /usr/lib/pd/Gem/Gem.pd_linux 
	Copia Gem.pd_linux a /usr/lib/pd/Gem

9. Vuelve al directorio donde descomprimiste el PMS y descomprime la estructura de directorios de los media (estructura-media.tar.gz) en el directorio que quieras. Abre config.pd y cambia el path por el del directorio de tus media. Alternativamente crea una estructura de directorios que PMS pueda entender:

- Números del 0 al 255: Videos, dentro de cada directorio los videos deben de estar numerados también del 0 al 255 y con la extensión .mov
- Directorio "imagenes"; PMS espera encontrar ficheros de imagen nombrados de 0 a 255 con extensión .jpg
- Directorio "fonts"; PMS espera encontrar ficheros de fuentes True Type numerados de 0 a 255 y con extensión .ttf	

10. Arrancando main.pd con pure data extended ya debería de estar funcionando. Dentro del programa puedes configurar más cosas como las direcciones DMX y textos para mostrar. Ver manual.txt.
